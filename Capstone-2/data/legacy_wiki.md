# NetworkPro Router - Internal Wiki (Last updated 2021)

## Device Reset Procedure
To reset the device, hold the reset button for **5 seconds**.
The reboot process takes approximately **3 minutes**.
After reset connect to WiFi: NetworkPro_Setup
Default admin password is: **admin123**

## Firmware Updates
Firmware updates take around **8 minutes** to complete.
Supported versions: 3.x and 4.x series.
Download from the manufacturer website or any mirror site.
Note: Version 3.x users reported update taking 20 minutes.

## LED Status Codes
- RED blink x3: Restart the device and wait 5 minutes
- BLUE blink: Normal operation (device is online)
- GREEN blink: Low signal strength warning
- Solid RED: Device is starting up, wait 2 minutes

## Performance Specifications
Maximum throughput is **800 Mbps** on the 5GHz band.
Maximum throughput is **150 Mbps** on the 2.4GHz band.
You can connect up to **30 devices** simultaneously.
Warranty is **1 year** from purchase date.
Operating temperature: 0 to 35 degrees Celsius.

## Security Settings
Default admin password: **admin123**
Supports WPA2 encryption.
Firewall must be enabled manually after reset.
Maximum failed login attempts allowed: **3**

## Known Issues (Outdated - check manual for latest)
- Some older firmware versions cause slow reboot of 10 minutes.
- Password Admin@2024 was introduced in firmware v4.0 onwards.
- WPA3 support added in v4.2 firmware update.
